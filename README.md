## Installation

Clone the project:

```bash
git clone git@github.com:VicaSpace/vicaspace.git
```

or

```bash
git clone https://github.com/VicaSpace/vicaspace.git
```

Navigate to the project directory:

```bash
cd vicaspace
```

Install yarn globally with npm:

```bash
npm i -g yarn
```

Install the dependencies:

```bash
yarn
```
